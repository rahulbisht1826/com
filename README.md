# Portfolio Website

This is a high-performance, SEO-optimized, and fully responsive portfolio website.

## Features
- **Premium UI**: Glassmorphism design with animated background globes.
- **Responsive**: Fully responsive from mobile devices to large desktop screens.
- **SEO Optimized**: Includes semantic HTML, meta tags, Open Graph tags, `robots.txt`, and `sitemap.xml`.
- **Fast**: Pure HTML/CSS/JS with no heavy frameworks.

## Setup
1. **Edit Content**: Open `index.html` and replace "Your Name" and the placeholder text with your actual details.
2. **Images**: 
   - Replace the project background gradients or add actual images in `style.css` (or inline styles in `index.html`).
   - Update `og-image.jpg` reference for social sharing.
3. **Run Locally**:
   - You can simply open `index.html` in your browser.
   - For a better development experience, use a local server (e.g., `npx serve` or Live Server in VS Code).

## Customization
- **Colors**: Edit the `:root` variables in `style.css` to change the color scheme.
- **Animations**: Adjust the `.background-globes` animation duration in `style.css`.

## Deployment
This static site can be deployed anywhere (GitHub Pages, Netlify, Vercel, Firebase Hosting, etc.) just by uploading these files.
